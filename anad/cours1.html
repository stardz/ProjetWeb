Introduction et définitions

Vous n'êtes pas sans savoir que JavaScript permet de lancer des fonctions lors de l'exécution d'événements. Ainsi,
dans notre expérience sous-marine, nous bougions l'arrière-plan lors de l'événement « passage de la souris sur les
flèches du tableau de bord ».
Les transitions CSS ont fait naître un nouvel événement : transitionend. Comme les anglophones parmi vous l'auront deviné,
cet événement se produit lorsque la transition s'est achevée.
Sur le papier, ça peut sembler pratique, notamment pour créer un système d'animations enchaînées. Malheureusement,
tout comme la propriété CSS, l'événement JavaScript change de nom selon le navigateur du visiteur.
